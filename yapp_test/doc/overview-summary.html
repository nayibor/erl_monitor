This application is for an opensource transaction monitoring system . This system receives iso 8583 messages from diverse sources on a tcp-ip server and forwards it to a websocket server for distrubition to user browsers whom are authorized to view those messages . main purpose of this application is to receive realtime feedback on status of transactions so that quick actions can be taken as opposed to polling of the database or other non real time means of monitoring . monitoring system will give visual and audio cues in case of important events and may support basic analytics
